PG3302 – Exercises for session 01
Note: It can be a good idea to work two or three together on these tasks: It’s nice to have someone
to discuss possible solutions with! Modelling can be done on paper or with online tools (google
"online UML diagram tool" etc.). You do not need Visual Studio today. (But do install it, see task 3!)
Task 1 - Associations for class diagrams
Consider the following associations, and choose whether they are aggregation or composition:
(also put the "diamond" on the right side)
a) Data file and the folder the file is in.
b) Student and a subject the student follows.
c) A car and its engine.
Task 2 – Implementation Class Diagram for an online store
This task can be done on paper. But if you have a UML tool available, you can use that too.
Note: If the tool you are using does not have 100% standard UML symbols, that’s ok: choose
something that resembles the symbols you are missing.
We will plan the code solution for a new online store. The following are features of the online store.
Note: The focus is to get some experience with UML class diagrams, symbols for this, etc. No
problem if you feel some of the domain logic is a little off. We will improve on that part throughout
the semester! :-)
Some keywords for content:
• Customer / user / account
• Goods
• Shopping cart
• Order
• Payment solution (credit card, PayPal, ...)
Set up the implementation class diagram for the online store solution. (Think about a real online
store and how it works.)
Regarding the size of this diagram, it is ok to make a small model (5-6 classes), there is no need to
create an implementation class diagram for a very large store system. (Note: If this were a real
project that we were going to implement, then it would have been relevant to have an
implementation class diagram for the entire solution.)
Task 3 – Install C# IDE (Visual Studio)
See today's lecture material for info and tips on choosing and downloading an IDE.
Visual Studio (Microsoft) is most widely used in the industry. Note: Also used by the lecturer.
(Visual Studio for Mac for those with macOS.) Other options are Visual Studio Code (also from
Microsoft) as a more lightweight alternative. Or Rider (from JetBrains) as a third option.
Note: For the course's exam hand-in, it is required that the project can be opened in Visual Studio!
Task 4 – Getting started with C#: Hello World
To get started with C#:
a) Create a new project of type "Console app".
b) Create your own version of Hello World, resembling what was done in the lecture. Anything
that lets you get started with C# coding is ok. :-)
c) Extend the application: Print a random name instead of "World". For example, the
application will print "Hello, Jens!" the first time and "Hello, Hans!" the second time.
(Without changing the code). Hint: Google "C# random", "C# list" and/or anything else you
think you might need.
